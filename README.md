# Breast Cancer Diagnosis Classification

Classification of breast cancer diagnoses as malignant or benign using machine learning models in R.

## Overview
Compared five classification models on biomedical imaging data from the UCI Machine Learning Repository:
- k-Nearest Neighbors (kNN)
- Logistic Regression
- LASSO Regularization
- Random Forest
- XGBoost with SHAP interpretation

## Results
Best performing model was kNN with 97.35% accuracy and AUC of 0.9935.

## Tools & Technologies
- R (caret, glmnet, xgboost, ggplot2)

## Data Source
- [UCI Breast Cancer Wisconsin Dataset](https://archive.ics.uci.edu/dataset/17/breast+cancer+wisconsin+diagnostic)
